# biblioteca-python

#  Sistema de Biblioteca em Python
Sobre
Sistema simples de gerenciamento de biblioteca desenvolvido em Python com SQLite.

##  Funcionalidades
- Cadastro de livros
- Listagem de livros
- Remoção de livros
- Cadastro de usuários
- Listagem de usuários
- Remoção de usuários

##  Tecnologias
- Python
- SQLite

##  Como executar
python biblioteca.py
